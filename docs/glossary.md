# Glossary of Terms

#### markdown
A lightweight markup language that you can use to add formatting elements to plaintext text documents.

#### material

A mkdocs deign them based on the [Material Components](#material-components) are interactive building blocks for creating a user interface.

#### material components

A set of interactive building blocks for creating a user interface.

See: [https://material.io/](https://material.io/)

#### mkdocs

A fast, simple and downright gorgeous static site generator that's geared towards building project documentation.

See: [mkdocs.org](https://www.mkdocs.org/)
